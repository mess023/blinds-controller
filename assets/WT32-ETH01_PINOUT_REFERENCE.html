🌐 WT32-ETH01 GPIO Pinout Reference

Motorised Window Blinds Controller — Pin mappings for the bottom & top stepper motors, limit switches, and driver control.

⚙ Stepper Motors

Motor Control Pins

GPIO Signal Function
GPIO 32 STEP Bottom blind stepper pulse input
GPIO 33 DIR Bottom blind stepper direction (HIGH=extend, LOW=retract)
GPIO 15 STEP Top blind stepper pulse input
GPIO 14 DIR Top blind stepper direction (HIGH=extend, LOW=retract)
GPIO 13 ENA Shared enable pin for both DM542T drivers — active LOW. Optocoupler: HIGH=disabled (no current), LOW=enabled (running)
Hardware connection: Wire ENA+ on both DM542T drivers to 3.3V (through ~1kΩ series resistor). Wire ENA- on both to GPIO 13. When GPIO 13 is HIGH, the optocoupler conducts and cuts motor current (idle power saving). When GPIO 13 is LOW, motors can run normally.
🔘 Limit Switches

Blind Position Limit Switches

GPIO Button ID Function
GPIO 2 Button 1 Bottom blind start-of-travel switch (fully retracted)
GPIO 4 Button 2 Bottom blind end-of-travel switch (fully extended)
GPIO 17 Button 3 Top blind start-of-travel switch (fully retracted)
GPIO 12 Button 4 Top blind end-of-travel switch (fully extended)
Debounce: All switches use 50ms debounce time in firmware to eliminate contact bounce noise.
⚠️ Motor Fault Monitoring

Driver Alarm Pin

GPIO Signal Function
GPIO 5 ALM Motor driver fault detection (pulled HIGH by internal pull-up, LOW when fault occurs)
Hardware connection: Wire ALM+ on both DM542T drivers to 3.3V (through ~1kΩ series resistor). Wire ALM- on both drivers together to GPIO 5. GPIO 5 is configured with INPUT_PULLUP. When either driver detects a fault (overcurrent, thermal, etc.), the ALM- pin pulls to ground, GPIO 5 reads LOW, and the firmware cuts power to both motors immediately.
🌐 Network & Power

Reserved Pins (Ethernet & Power)

The following pins are reserved for Ethernet, power delivery, and are not available for user application:

  • GPIO 0, 1 (Boot, serial)
  • GPIO 6–11, 16, 20–24 (Quad SPI for external flash)
  • GPIO 26, 27 (USB D+ / D−)
  • GPIO 25 (internal use)
  • GPIO 35–39 (GPI — input-only, no output)
